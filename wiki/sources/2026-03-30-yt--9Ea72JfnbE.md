# Jensen Huang's New Plan for Employees Is INSANE (Marketing School)

## Summary

A medium-density co-hosted episode in two clear halves. **Half one** is a news reaction to NVIDIA CEO Jensen Huang saying he'd "be deeply alarmed if a $500,000 engineer was not spending $250,000 worth of tokens." Both hosts agree AI should be used heavily, but push back on the *spend*: Neil argues most of the token budget should run through **open-source models on local infrastructure** (maybe 70–90%), so you get the same result for ~$30k instead of ~$250k, and openly frames Jensen as a "genius salesperson" whose pitch conveniently sells NVIDIA hardware. Eric shares his own AI spend (3.5 billion tokens/month, ~$5.6k on Anthropic) and both note internal AI cost is "exploding" into a "massive cost center" — the beneficiaries so far are Cursor and Anthropic, not their own companies. Digression on Cursor's valuation ($29.3B → reportedly $40B), $2B revenue run rate (Bloomberg), and vertical/"institutional" models (Intercom's **Fin**).

**Half two** is the durable marketing content: while Eric steps away, **Neil delivers a solo, numbered "SEO trends for 2026" monologue** (explicitly Neil — "I'll update Eric when he comes back"). Highlights: (1) go back to sites that already link to you and get them to add brand/product-name *mentions* (not just anchor text) — easier than convincing a random site, helps SEO and GEO; (2) SEO via API / CLI tooling (Ramp CLI, querying Mixpanel + Stripe, still human-in-the-loop); (3) rising brand search (Google Trends up-and-to-the-right) correlates with higher non-branded rankings, so brand campaigns help SEO indirectly; (4/AEO) their internal agent "Oracle" scores them ~80–85 on SEO but ~30 on AEO — run an AEO checklist (structured data, three key points up top, FAQ, citations) alongside SEO; (5) double down on **EEAT via human-written content** (AI for research/ideation/refresh only); (6) proprietary data/charts/surveys earn links and, shared to LinkedIn, get cited more by LLMs. Then agentic-commerce/checkout talk (Meta + Stripe ACP, AMP-for-email checkout, keeping buyers on-platform for conversion) and a low-density product-banter tail (Grüns gummies, David Beckham's IM8 bundle/checkout, Legion protein).

★ L3-candidate — the clearly-Neil "SEO trends 2026" framework (linker-mentions, brand→non-branded ranking signal, AEO checklist, EEAT/human-written, proprietary-data→LinkedIn/LLM citations) is a coherent, reusable, firmly-stated marketing framework. The AI-token/Cursor half is news-reactive and ephemeral — do NOT promote that portion.
