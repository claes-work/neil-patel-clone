# Don't Build Backlinks This Year - Do This Instead

## Summary
Argues that, especially for budget/time-constrained sites, writing one high-quality
piece of content per week for a few months beats active link building — while stressing
he is not against link building in general, only against ever paying for links.
Introduces Ubersuggest's "page one ranking potential" feature (under keyword ideas):
enter a URL and it scores which keywords the site could realistically rank for given
its current on-page SEO, backlink profile, and site history; write content targeting
those keywords, grouped into topic clusters (his example: a beginner's-guide-to-SEO hub
with on-page SEO, link building, page speed, and AMP as separate cluster articles).
Repeats the "audit page 1 vs. page 3-4 of Google" competitive-gap exercise from the
2021-10-21 video. Closes with the "put the user first" doctrine already documented in
the wiki: users care whether a result solved their problem quickly, not how many links
a page has; once a content foundation exists, add links later to push into more
competitive terms.
