# Ranking Your eCommerce Store On Google - Module 2 - Part 2 - eCommerce Unlocked

## Summary
eCommerce Unlocked lesson on SEO + content production for e-commerce sites. Opens
with **mobile-first indexing**: Google maintains separate desktop and mobile indexes,
so responsive design (not a separate mobile app) matters directly for traffic, checked
via Ubersuggest's site audit "green" mobile-load-time targets; shorter, less-cluttered
copy is recommended for mobile readability. Walks the Ubersuggest workflow for ranking
product pages: Keyword Ideas report (search volume, "SD"/SEO-difficulty, CPC as a
higher-intent signal) plus its Related/Suggestions/Questions/Comparisons tabs;
Traffic Analyzer's Keywords report to see what a domain (own or competitor's) already
ranks for; Site Audit for on-page/meta/backlink-quality issues. On-page checklist:
titles, bullets, descriptions, keywords, category-page optimization, internal
cross-linking product pages ↔ related products ↔ About/Details pages ("don't just
link for the sake of SEO... if it doesn't [help the experience], then don't add that
link"), Google Shopping listing (free). Content-production workflow (using Trello):
research (Ubersuggest Content Ideas report, sorted by estimated visits/backlinks/
social shares) → outline (~45 minutes hook/body/close) → draft → edit → publish →
outreach (email everyone cited, asking for a share; use the Backlinks report on
competitor URLs to find outreach targets). Case study again: Skincare by Alana's
product pages (video + image + text depth) as the exemplar.

## Key claims
- As of 2020-07-02, Neil said Google runs separate desktop and mobile indexes, and
  recommends responsive design (not a separate mobile site/app) verified via
  Ubersuggest's mobile-load-time "green" scoring — consistent with existing
  mobile-first doctrine already in the wiki.
- As of 2020-07-02, Neil's content-outline time-box is ~45 minutes covering hook,
  body, and close, done in Google Docs or Microsoft Office before handing to editors.
- As of 2020-07-02, Neil's internal-linking philosophy is experience-first, not
  SEO-first: "don't just link for the sake of SEO, but link because it provides a
  better experience. And if it doesn't, then don't add that link."
- As of 2020-07-02, Neil cited a Bluestout quote as validation for competitive
  benchmarking: if a competitor's product page consistently outranks yours, "that's
  proof that you need to work on your product page" (external attribution, not his own
  coinage).
