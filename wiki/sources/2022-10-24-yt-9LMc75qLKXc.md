# How Google Wants You to Create Meta Descriptions

## Summary
Paraphrases Google's own meta-description guidance (don't keyword-stuff, don't reuse
one description site-wide, don't just summarize the page, don't make it too short) as
a baseline that "helps you" but "doesn't really give you enough information to
separate yourself from the crowd" — Google's job is user experience, not teaching
sites how to win. Presents Ubersuggest-derived click-through-rate data (tool tracks
"over 900 million domains") as the differentiator: descriptions of ~119-135 characters
get roughly +3.4% CTR; including the searcher's exact query in the description adds up
to +5.7%; "power words" (example given: "effortless") add roughly +2.18%; descriptions
that convey emotion add roughly +2-3%; and simply having ANY meta description (versus
none) adds roughly +5.8% CTR versus letting search engines auto-generate one. Frames
these gains as individually small but additive when several tactics are combined
("all those small percentages will add up"). Closes noting NP Digital holds similar
proprietary data for title tags, URLs, on-page SEO, and link building, with an agency
plug.

## Key claims
- Cites Ubersuggest's tracked corpus as "over 900 million domains" (self-reported tool
  scale claim, dated 2022-10-24).
- States meta descriptions of ~119-135 characters get "roughly a 3.4% higher"
  click-through rate versus too-short/too-long descriptions (Ubersuggest-derived
  proprietary data, self-reported).
- States including the searcher's exact query in the meta description yields "up to a
  5.7% increase" in CTR (Ubersuggest-derived, self-reported).
- States "power words" (example: "effortless") increase CTR "by roughly 2.18%"
  (Ubersuggest-derived, self-reported).
- States emotional meta descriptions yield "roughly a 2 to a 3% higher" CTR than
  non-emotional ones (Ubersuggest-derived, self-reported).
- States simply having a meta description (regardless of content/length/keywords)
  yields "roughly a 5.8% higher" CTR than having none at all (Ubersuggest-derived,
  self-reported).
