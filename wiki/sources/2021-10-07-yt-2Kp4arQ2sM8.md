# A Realistic Approach to SEO For Busy Solo Entrepreneurs

## Summary
A time-boxed SEO playbook aimed at solo operators with no team, split into optimizing
existing content and creating new content. Optimizing: use Google Search Console to
find high-impression/low-CTR pages and improve title tags/meta descriptions with
persuasive (not clickbait) copy using the same keywords GSC surfaces; use Ubersuggest's
rank-tracking and content-ideas reports to spot momentum shifts and headline ideas;
use Google Analytics to find underperforming pages similar to strong pages and 301-
redirect/consolidate them (explicitly: backlink count alone doesn't matter if traffic
doesn't follow — redirect the higher-backlink loser into the higher-traffic winner).
Creating new content: use Ubersuggest's keyword-ideas report (with the "probability of
ranking on page one for your domain" feature) and competitive-analysis report (top
pages, social shares, ranking keywords, and their linkers to pitch) to build topic
clusters instead of single pages. Closes with concrete time budgets: on-page technical
setup (CDN, cache, image compression, AMP, sitemap, robots.txt) is a one-off 2-day–
2-week project with quarterly/annual tune-ups; new content creation should run 2–8
hours/week depending on niche competitiveness; editing/optimizing existing content
should run 4–10 hours/month.

## Key claims
- Gives concrete solo-operator time budgets: 2 days–2 weeks for one-off on-page
  technical setup, 2–8 hours/week for new content creation, 4–10 hours/month for
  editing/optimizing existing content (self-reported, dated 2021-10) — seo/
  content-marketing. New quantified time-allocation figures not previously documented
  (distinct from the earlier-documented 10/40/20/30 content-time-allocation doctrine).
- States that backlink count alone is not a redirect/consolidation criterion — traffic
  is; a page with 100 backlinks but 2 visitors/month should be redirected into a
  similar page with 10 backlinks but hundreds of visitors (paraphrase, dated 2021-10)
  — analytics-cro/seo. Concrete tactical elaboration of the existing
  content-consolidation doctrine.
- Recommends targeting "at least 50% better" content than the current top-ranking
  page as the bar for new content, rejecting "me too" content (paraphrase, dated
  2021-10) — content-marketing. Reinforcing.
