# How To Get The Most Out Of Google Analytics - Module 06 - Lesson 1 - SEO Unlocked

## Summary
SEO Unlocked module 6 opener ("Mastering Google Tools"), lesson 1 of 3, on Google
Analytics fundamentals. Opens with an Oscar Wilde quote framing analytics as an error-
finding tool, then a checkpoint: after five modules, viewers should already be seeing
measurable results (points to the downloadable "SEO Taskmaster Worksheet" if not).
Four setup steps: (1) **Goals** — Admin → Goals → pick a template (purchase, lead,
contact) matching the site's conversion action, most simply a destination goal (a
thank-you-page URL), then confirm the goal is turned ON. (2) **Internal search
tracking** — Admin → Settings → Site Search Tracking, to see what visitors search for
on-site as a source of content ideas / unmet-offer signals. (3) **Custom dashboards**
— Customization → Dashboards → Create, built from a blank canvas with only
revenue-related metrics ("I wouldn't create a dashboard with a ton of vanity
metrics... vanity metrics don't move the needle, don't make you money"), laid out
split-screen. (4) **Custom alerts** — sidebar → Create Custom Alerts → New Alert, to
get notified of traffic/goal/revenue/bounce-rate spikes or drops without having to
check Analytics constantly. Action items: set up a dashboard, create goals, download
the SEO Taskmaster Worksheet.

## Key claims
- As of 2020-06-06, Neil framed Google Analytics' core value as hypothesis
  formulation — distinguishing content/keywords that build links and social shares
  from content/keywords that drive revenue vs. just traffic.
- As of 2020-06-06, Neil said custom dashboards should be built only from
  revenue-related metrics, explicitly rejecting "vanity metrics" as dashboard content
  because they "don't move the needle, don't make you money."
- As of 2020-06-06, Neil recommended internal site-search tracking (Admin → Settings →
  Site Search Tracking) as a source of content and product-offer gaps — if visitors
  search for something the site doesn't have, that's a signal to create/offer it.
- As of 2020-06-06, Neil recommended custom Analytics alerts (traffic spikes/drops,
  goal completions, revenue, bounce rate) as a way to stay informed without manually
  checking Analytics daily.
