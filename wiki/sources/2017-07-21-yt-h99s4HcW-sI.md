# Are SEO Backlinks From Article Directories Bad For Google Search Rankings?

## Summary
Neil's answer to whether article-directory links help SEO is a flat **no** for ranking purposes. Because directory links are so easy to get (many free, some paid), Google assigns them little value regardless of the directory's Domain Authority. He reports building hundreds of them himself; they worked ~6–7 years earlier but stopped around 2013–2014. His nuance: existing directory links don't need to be disavowed (they neither help nor hurt), and there's one legitimate reason to use directories — **business referrals**, not rankings (e.g. legal directories like Avvo or FindLaw for lawyers). Rule: build directory links only if they drive business, never to manipulate Google.

## Key claims
As of 2017-07-21, Neil states:
- Article-directory links are one of the easiest links to get, and Google knows it — so it doesn't place much value on them even if the directory has a Domain Authority of 70 or 80 (as of 2017-07-21).
- He personally built hundreds of article-directory links; they worked well ~6–7 years ago but stopped working around 2013–2014 (as of 2017-07-21).
- He would not build any more directory links for SEO; existing ones from the past are okay and don't need to be disavowed (as of 2017-07-21).
- Disavowing = telling Google not to index/count those links toward yours; article-directory links "don't really hurt you but they don't really help you," so don't waste time on them (as of 2017-07-21).
- The only reason to build directory links is if it can **drive business** — e.g. legal directories like Avvo and FindLaw can bring a lawyer real clients (as of 2017-07-21).
- Those business-driving directory links will **not** help search rankings; use them for referral traffic, never to manipulate Google (as of 2017-07-21).

## Notable verbatim quotes
