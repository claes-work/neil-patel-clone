# The Truth About Video Views on X (Twitter)

## Summary
Prompted by Eric's question comparing a Twitter/X view to a YouTube view, Neil
explains X counts a view from a passive autoplay-while-scrolling (no real engagement
required), whereas YouTube requires a few seconds of actual engaged playback before
counting — making an X view worth far less than a YouTube view. Despite that, Neil
argues it's still worth cross-posting the same video to X: since the content-creation
work is already sunk cost, posting to another platform is nearly free, and any extra
views (as long as the audience is roughly relevant) are pure upside. He adds that
some platforms (in his case, Twitter/X) are more useful for learning from smart
people than for reach, and that view performance is unpredictable across platforms —
sometimes Instagram outperforms, sometimes TikTok has a viral hit — so cross-posting
everywhere is worth the near-zero marginal cost.

## Key claims (dated, paraphrase unless noted)
- Platform-view-definition claim: an X/Twitter view counts even from a passive
  autoplay-while-scrolling with no real engagement; a YouTube view requires roughly a
  few seconds of actual engaged playback — making X views far "cheaper"/less
  meaningful than YouTube views. (Neil, 2023-11-16) — social-media.
- Core recommendation: cross-post the same video content to X (and by extension other
  platforms) even if reach/quality-per-view is lower, because the hard work
  (creating the content, the hook, the idea) is already done — posting elsewhere is
  near-zero marginal cost, so any extra relevant views are pure upside.
  (Neil, 2023-11-16) — social-media / content-marketing.
- Notes cross-platform performance is unpredictable — sometimes Instagram
  outperforms, sometimes a TikTok post unexpectedly goes viral — arguing this
  variance itself justifies posting everywhere rather than picking one channel.
  (2023-11-16, attribution uncertain between Neil/Eric — likely Neil given the
  self-referential framing) — social-media [attribution: uncertain].
