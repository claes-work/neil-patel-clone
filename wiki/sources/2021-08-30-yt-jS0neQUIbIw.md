# How To COMPLETELY RECOVER Your Lost SEO Traffic

## Summary
A traffic-recovery checklist, presented as NP Digital's own client-facing "formula."
Steps: (1) check whether a traffic drop correlates with a known Google algorithm
update (via a penalty-indicator tool + Search Engine Roundtable) versus a manual
penalty (Google Search Console). (2) Run an Ubersuggest Site Audit for prioritized
technical fixes. (3) Check for malware/hacked content (a `site:` search plus Sucuri's
sitecheck) — Neil notes NP Digital has "sometimes" seen bad/spam links (adult,
gambling, pill-site anchor text) land on client sites and recommends disavowing them,
though he notes Google usually ignores unnatural links a site didn't create itself.
(4) Run a link audit via Ubersuggest for shady referring domains/anchor text.
(5) Use GSC's date-comparison (last 28 days vs. a year/months ago) to find keywords
that used to rank, then competitively re-diagnose why (depth vs. conciseness — the
recurring "how to tie a tie" example). (6) Check for keyword cannibalization in GSC
and consolidate weaker duplicate pages via 301 redirect + internal-link cleanup
(Screaming Frog). (7) Prune irrelevant/thin content — reprises the About.com
domain-split/page-deletion case study (already in the wiki) as evidence that fewer,
stronger pages beat a large weak-content footprint.
