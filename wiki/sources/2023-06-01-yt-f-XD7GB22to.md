# How to start using TikTok Ads to supercharge your e-commerce business

## Summary
Short solo scripted tutorial (no interview/guest content) walking through a four-step TikTok
Ads setup checklist for e-commerce advertisers new to the platform. **(1) Daily budget**: set
at least $30/day (or local-currency equivalent) minimum, framed as the floor needed for
TikTok's algorithm to learn the target audience and reach optimal performance during testing.
**(2) Spark Ads**: TikTok's native format that converts existing organic TikToks (the
advertiser's own, or another creator's with permission) into paid ads on the For You feed;
recommends this over traditional static/produced ad creative because Spark Ads read as
organic content and "generally perform better" — cites (paraphrasing TikTok's own internal
research, attributed to the platform, not verified independently) that the best-performing
Spark Ads feature videos closely tied to the actual product, ideally recent ones, though any
existing TikTok can be used. **(3) Signals and the TikTok Pixel**: defines "signals" as any
tracked customer interaction (video views, site visits, purchases) matched to a TikTok user,
and frames the Pixel as the mechanism that unlocks them; gives a four-step Pixel setup
checklist — install the Pixel; configure at least three events (recommends view-content,
add-to-cart, complete-payment specifically); enable automatic advanced matching; and confirm
first-party cookies are enabled in Events Manager — with a note that developer-assisted
manual advanced matching plus the Events API perform better still, and that Shopify/
WooCommerce integration handles pixel/event setup in a few clicks for merchants on those
platforms. **(4) Mid-funnel optimization**: recommends new advertisers optimize their first
conversion campaign toward the **add-to-cart** event rather than jumping straight to
complete-payment, on the logic that the algorithm needs to be taught what a likely-to-convert
visitor looks like before it can effectively target purchase-stage intent; gives an explicit
escalation rule — if a campaign generates an add-to-cart event within the first week, launch
a second ad group targeting complete-payment; if it does not, step back further to a
view-content-optimized ad group until reaching 100 view-content events, then move down-funnel.
Additional tactical notes: keep ad groups "always on" where possible to preserve account-level
learning; start with broad/auto targeting before layering custom-audience retargeting.
