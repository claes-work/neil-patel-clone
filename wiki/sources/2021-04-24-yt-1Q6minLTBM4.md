# How to Get More Backlinks (FAST)

## Summary
Solo tactical screen-recorded tutorial demonstrating Ubersuggest's Backlinks Checker as a
fast link-building workflow, built on two techniques. (1) **Link reclamation**: run the
"new and lost backlinks" report on your own domain (his own neilpatel.com shows 3.3M+
tracked backlinks, 79 lost the day of recording), manually vet each lost link (some are
legitimate sites worth reclaiming, e.g. cocolyze.com which used to link to Ubersuggest;
others are spammy junk to ignore), find the site's contact email via hunter.io (free tier:
25 searches/month), and send a short, specific outreach email asking why the link was
removed and inviting them to re-add it, citing new features shipped since. (2)
**Competitor link mining**: run the same new/lost report on a competitor (demoed on Moz)
but focus on their *newly gained* links rather than lost ones — recent link activity is
evidence the linking site is actively updated and currently link-friendly — filter for
topical relevance (e.g. skip an unrelated "Hummingbird Barbecue" post that only matches
because Moz's "Hummingbird" algorithm update shares the name), then pitch inclusion the
same way (hunter.io email lookup + short personalized ask). Recommends running the whole
process weekly (or up to daily, 30 min/day) as a sustainable outreach cadence.

## Key claims
- Recommends prioritizing your OWN recently-lost backlinks for reclamation outreach over
  cold prospecting, since these are sites that already found you valuable enough to link
  to once (paraphrase, 2021-04-24) — seo.
- On competitor-based link mining, explicitly argues you should target the competitor's
  **newly gained** links, not their lost ones, because a "hey, they stopped linking to my
  competition, so start linking to me" pitch "doesn't really work well" — a site's *new*
  links are the better signal of current link-friendliness (paraphrase/near-verbatim,
  2021-04-24) — seo.
- States neilpatel.com has over 3.3 million backlinks tracked in Ubersuggest's Backlinks
  Checker, with 79 lost on the day of recording (self-reported, tool-derived) — seo.
- Recommends doing this link-building process on a weekly cadence at minimum (an hour a
  week), or daily in 30-minute sessions if more time is available, applied across all of
  a site's competitors (paraphrase, 2021-04-24) — seo / agency-entrepreneurship.
- Recommends hunter.io (25 free searches) as the tool for finding a target site's contact
  email address for outreach (self-reported tool recommendation, 2021-04-24) — seo.
- Notes that outreach emails should be short, specific (cite the exact URL and what
  changed), and low-pressure — acknowledges some recipients will ignore the email, some
  will explain why they removed the link (useful feedback), and some will simply re-add
  it (paraphrase, 2021-04-24) — seo.
