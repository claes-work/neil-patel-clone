# How Much Time Do You Need to Invest In SEO With a New Website That Has No Traffic

## Summary
A concrete weekly-time-budget framework for new/zero-traffic sites: don't duplicate
content on the same topic; pivot toward less-competitive long-tail keyword variations
via Ubersuggest's keyword-ideas report when a head term isn't ranking; use Ubersuggest's
domain-specific "keywords you could rank page one for" filter from day one; once
traffic starts, use Google Search Console to find high-impression/low-click pages and
add those keywords into title tags and meta descriptions. Gives an explicit time
budget: content writing ~4 hours/week, keyword research ~2-3 hours/month. Bonus tip:
run Ubersuggest's site-audit report monthly (not weekly) and fix issues top-to-bottom
by impact; setting up an Ubersuggest "project" automates rank tracking and recurring
site-audit checks.

## Key claims
- Explicit weekly/monthly time budget for a new site's SEO: ~4 hours/week on content
  writing (research + writing), ~2-3 hours/month on keyword research (self-reported,
  dated 2021-11-08) — seo, analytics-cro. A more specific, smaller time figure than the
  "2-8h/wk content, 4-10h/mo editing" range already logged from the 2021-10-07 video —
  both are self-reported estimates for solo/small operators, not in direct conflict but
  worth noting as a range rather than a fixed number.
- Content-duplication warning: don't write near-duplicate articles on the same topic
  ("Beginner's Guide" vs. "Ultimate Guide for Beginners" example) — if a topic isn't
  ranking, pivot to a genuinely different, less-competitive long-tail variant instead
  (paraphrase, dated 2021-11-08) — content-marketing, seo.
- Recommends running Ubersuggest's site-audit report once a month (not more often),
  fixing issues in the tool's own priority order (paraphrase, dated 2021-11-08) — seo.
  Reinforces the existing Ubersuggest site-audit documentation with a monthly cadence
  detail.
- Ubersuggest's "project" feature automates recurring rank tracking and site-audit
  checks, surfacing new errors weekly without manual re-running (self-reported tool
  feature, dated 2021-11-08) — seo.
